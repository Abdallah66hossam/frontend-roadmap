2023 - Frontend Roadmap:

How the Internet works

As a frontend developer, you must have a good understanding of how the Internet works and networks and what is happening behind the scenes while opening a new website

HTML

HTML5 (Hypertext Markup Language 5) is a markup language used for structuring and presenting content on the World Wide Web. understanding what is semantic html and why to use it.

CSS

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. learning Responsive Layouts is essential.

CSS Framework

You don't need to build every thing from scratch so learning css framework will help you. TailwindCSS or Bootstrap are good options

JavaScript

JavaScript is a scripting or programming language that allows you to implement complex features on web pages ( api request from the server, popup alerts, etc ).

GIT & GitHub

Git is a version control system that tracks changes in any set of computer files, used for coordinating work among programmers who are collaboratively developing source code during software development. And GitHub is the place where you can store your code.

React.js

React.js is a JavaScript library for building user interfaces, and it is widely used in web development today. you can learn other libraries like Angular or Vue.js also.

TypeScript

TypeScript is a typed superset of JavaScript that compiles to JavaScript, it provides many features that can help you write more robust, maintainable, and scalable code.

Unit Testing (Jest)

Jest is a popular JavaScript testing framework that is widely used for testing JavaScript and React applications.
